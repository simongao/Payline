# 交易记录的导出

## 中信证券通达信交易软件

1. 在交易软件中导出**资金流水**，选择Excel格式。如果有融资融券账户也要导出。注意通达信一次只能查询一个月的交易记录，需要逐月导出。
2. 使用本项目中包含的MergeLog.py工具，将单月的交易记录进行合并。
3. 用Excel打开合并的.xls文件，并另存为标准的.xlsx文件。
